问题描述

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1

输入: 121
输出: true

示例 2

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

示例 3

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。

进阶:

你能不将整数转为字符串来解决这个问题吗？

思路

将数字一半反转，然后将反转后的数字与原始数字另一半进行比较，如果它们是相同的，那么这个数字就是回文。 例如，输入 1221，我们可以将数字“1221”的后半部分从“21”反转为“12”，并将其与前半部分“12”进行比较即可。 负数不是回文。

实现

func isPalindrome(x int) bool {
	if x < 0 {
		return false
	}
    
    if (x < 10) {
        return true
    }


    p, k := 0, x
    for k!=0 {
        n := k % 10
        k/=10
        p = p*10 + n
    }
	return p == x
}
